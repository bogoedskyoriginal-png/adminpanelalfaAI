# Alfa AI Notifications Demo

Локальный демо-проект без backend: админка + пользовательский интерфейс, чтобы создать уведомления и баннеры и увидеть поведение требований.

## Запуск

Откройте `index.html` в браузере.

## Что реализовано

- Центр уведомлений: список, статусы «Новое/Прочитано», индикатор новых.
- Ошибка загрузки: тумблер симуляции + кнопка повторной загрузки.
- Важные уведомления: модалка при первом входе после публикации, закрытие по крестику/кнопке.
- Адресная рассылка по `user_id` с валидацией.
- Контуры «в контуре / вне контура».
- Баннеры: период отображения, включение, баннер по умолчанию.

## Хранение

Состояние хранится в `localStorage` ключ `alfa-ai-demo-state`.
